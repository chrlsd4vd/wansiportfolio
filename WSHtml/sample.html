HTML (HyperText Markup Language)

Definition

HTML is the standard language used to create and structure web pages. It uses tags to organize content such as text, images, and links.

📁 File extensions .html , .htm

Basic HTML Structure

Every HTML document follows this structure:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

</body>
</html>

Explanation of each tag:

Tag Purpose
<!DOCTYPE html> Declares the document as HTML5
<html> Root element of the webpage
<head> Contains metadata (title, links, styles, scripts)
<title> Title shown in the browser tab
<body> Contains the visible content of the webpage

Common HTML Elements

Headings

HTML provides 6 heading levels.

<h1>Heading</h1>
<h2>Heading</h2>
<h3>Heading</h3>
<h4>Heading</h4>
<h5>Heading</h5>
<h6>Heading</h6>

📌 h1 = largest   📌 h6 = smallest

Paragraph

<p>This is a paragraph.</p>

Used to display blocks of text.

Link

<a href="https://example.com">Visit Website</a>

Attributes: href = destination URL.

Image

<img src="image.jpg" alt="Description">

src = image source, alt = alternative text.

Line Break

<br>

Used to move text to the next line.

HTML Attributes

Attributes provide additional information about elements. Example:

<img src="cat.jpg" alt="Cute cat">

Common attributes:

Attribute Description
id Unique identifier for an element
class Used to group elements for styling
src Specifies image source
href Specifies link destination
alt Alternative text for images
type Defines input type

HTML Forms – Input Types

Forms collect user input. Example: <input type="text">

Common input types:

Input type Purpose
text Text input
email Email address
password Hidden password input
number Numeric input
checkbox Multiple selection
radio Single selection
submit Submit button

HTML Lists

Unordered List (bullet)

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

📌 Displays bullet points
Example: • Apple • Banana • Mango

Ordered List (numbered)

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

📌 Displays numbers
Example: 1. Step One 2. Step Two

CSS (Cascading Style Sheets)

Definition

CSS is used to design and style web pages. It controls: Colors, Layout, Fonts, Spacing, Positioning.

Ways to Apply CSS

1. Inline CSS

<p style="color:red;">Hello</p>

📌 Applies style to one element only.

2. Internal CSS

<style>
p { color: red; }
</style>

📌 Used for single webpage styling (inside <head>).

3. External CSS

<link rel="stylesheet" href="style.css">

Example in style.css:

p { color: red; }

📌 Best practice for large websites.

CSS Syntax

selector {
  property: value;
}

Example:

p {
  color: blue;
}
Part Meaning
selector HTML element to style
property Style type
value Style value

CSS Selectors

Selectors target HTML elements.

Element selector

p { color: red; }

Class selector .box

.box { background: blue; }

HTML: <div class="box"></div>

ID selector #header

#header { background: black; }

HTML: <div id="header"></div> 📌 ID must be unique

Group selector

h1, h2, p { color: black; }

Advanced Selectors

Descendant selector

div p { color: blue; }

Meaning: all <p> inside <div>.

Child selector

div > p { color: red; }

Selects direct child elements only.

Adjacent sibling selector

h1 + p { color: green; }

Selects the first <p> after <h1>.

General sibling selector

h1 ~ p { color: purple; }

Selects all <p> after <h1>.

Pseudo Classes

Used for special states. Example:

a:hover { color: red; }
Pseudo‑class Meaning
:hover When mouse is over element
:active When clicked
:focus When input is selected

Pseudo Elements

Style specific parts of elements. Common pseudo‑elements:

p::before
p::after
::backdrop

🔹 ::before – inserts content before an element's content

Example:

p::before {
  content: "Note: ";
  color: #1d7a9c;
  font-weight: 600;
}

This paragraph has a ::before pseudo-element.

Each paragraph in this box gets a "Note:" prefix.

Output: The text “Note: ” appears before every paragraph inside the box above.

🔹 ::after – inserts content after an element's content

Example:

p::after {
  content: " 🔚";
  color: #b34a6c;
}

This paragraph has a ::after pseudo-element.

Each paragraph ends with a decorative arrow.

Output: A " 🔚" symbol appears after each paragraph above.

🔹 ::backdrop – styles the background behind a fullscreen element or <dialog>

Example (with a modal dialog):

dialog::backdrop {
  background: linear-gradient(145deg, #4a2b7a, #2b1b4a);
  opacity: 0.9;
  backdrop-filter: blur(4px);
}

✨ Hello! ✨

This dialog has a custom ::backdrop with a purple gradient and blur.

Combining ::before and ::after

p::before {
  content: "⭐ ";
}
p::after {
  content: " ⭐";
}

This paragraph gets stars on both sides.
